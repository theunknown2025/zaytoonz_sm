# Zaytoonz - Social Impact Community

Connect with Zaytoonz community - Fueling Social Impact with Professional Expertise.

## 🌟 About

Zaytoonz is a social impact organization dedicated to making a difference through professional expertise and community engagement. Our platform provides easy access to all our social media channels and community groups.

## 🚀 Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Cards**: Hover over social media cards to reveal QR codes for easy mobile access
- **Background Slideshow**: Dynamic background showcasing our focus areas (Health, Water, Green, Education)
- **Modern UI**: Clean, professional design with smooth animations

## 📱 Connect With Us

- **WhatsApp**: Community Chat
- **Telegram**: Official Channel
- **Facebook**: Official Page & Community Group
- **LinkedIn**: Company Page & Networking Group

## 🛠️ Technology Stack

- Pure HTML5
- CSS3 (with modern features like Grid, Flexbox, CSS Variables)
- Vanilla JavaScript
- Google Fonts (Poppins)

## 🌐 Hosting

This project is ready to be hosted on any static hosting service:

### Recommended Hosting Platforms:

1. **Netlify** (Recommended)
   - Drag and drop deployment
   - Free SSL certificate
   - Automatic deployments from Git
   - [Deploy to Netlify](https://www.netlify.com/)

2. **Vercel**
   - Zero configuration
   - Free SSL
   - Excellent performance
   - [Deploy to Vercel](https://vercel.com/)

3. **GitHub Pages**
   - Free hosting for public repositories
   - Custom domain support
   - [GitHub Pages Guide](https://pages.github.com/)

4. **Cloudflare Pages**
   - Fast global CDN
   - Free SSL
   - [Deploy to Cloudflare](https://pages.cloudflare.com/)

### Quick Deployment Steps:

#### Option 1: Netlify (Easiest)
1. Go to [Netlify](https://www.netlify.com/)
2. Drag and drop your project folder
3. Your site is live!

#### Option 2: GitHub Pages
1. Create a GitHub repository
2. Push your code to the repository
3. Go to Settings > Pages
4. Select your branch and save
5. Your site will be live at `https://yourusername.github.io/repository-name`

#### Option 3: Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in your project directory
3. Follow the prompts
4. Your site is deployed!

## 📝 Customization

### Update Social Media Links
Edit the `href` attributes in `index.html` for each social media link.

### Change Colors
The main color scheme uses olive green tones. To change:
- Primary green: `#a8d94a`
- Dark green: `#556B2F`
- Update these values in the `<style>` section

### Add/Remove Social Platforms
Duplicate or remove the social link blocks in the HTML and adjust the grid layout if needed.

### Update Background Images
Replace the images in the `public/` folder with your own images (keep the same names or update the CSS references).
